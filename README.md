## Description

Serverless API in Nestjs using Fastify, with swagger documentation.

# Libraries

- NodeJS
- Fastify
- Swagger
- Typescript
- Serverless
- aws-lambda-fastify
- aws-lambda

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run lambda:start:dev

# production mode
$ npm run lambda:start:deploy
```

## Swagger

`http://localhost:3000/dev/api/nest-serverless-fastify/docs`
